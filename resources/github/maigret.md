# Maigret: Advanced OSINT Username Search Tool

## Introduction
Maigret is an advanced OSINT tool designed to collect a dossier on a person by username across thousands of sites (3000+).

## Installation
### 1. Local Installation (via PyPI)
```bash
pip3 install maigret
```

### 2. Using Docker
```bash
docker pull soxoj/maigret
```

### 3. Windows Standalone
Download the EXE from the [Releases](https://github.com/soxoj/maigret/releases) page.

## Key Features
- **Massive Coverage**: Supports over 3,000 sites.
- **Deep Analysis**: Parses profile pages for personal info, bios, and links.
- **Recursive Search**: Discovers new usernames and IDs automatically.
- **Interactive Web UI**: Includes a graph-based visualization (`--web` flag).
- **Multiple Formats**: Reports in HTML, PDF, CSV, and JSON.
