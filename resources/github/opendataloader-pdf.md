# OpenDataLoader PDF: AI-Powered PDF to Structured Markdown/JSON/HTML

## Introduction
[OpenDataLoader PDF](https://github.com/opendataloader-project/opendataloader-pdf) is an open-source, AI-powered document processing system designed to convert complex PDF files into structured, AI-ready formats such as Markdown, JSON, and HTML. It is highly suitable for retrieval-augmented generation (RAG) and document intelligence pipelines.

## Key Features
- **Structured Extraction**: Converts PDFs into Markdown, JSON (containing bounding box metadata for precise citations), and HTML.
- **Hybrid Engine**: Combines fast, deterministic local parsing for standard text with AI-powered models for complex elements (multi-column layouts, scanned documents, tables, etc.).
- **Privacy & Security**: Operates completely locally, ensuring sensitive documents (legal, medical, financial) never leave the user's environment.
- **Advanced OCR & Table Extraction**: Built-in support for OCR (80+ languages), complex table parsing (including merged cells), mathematical formula recognition, and chart analysis.
- **Accessibility Automation**: Auto-tags untagged PDFs to generate screen-reader-ready Tagged PDFs.
- **Framework Integrations**: Officially integrates with common AI and LLM frameworks such as LangChain and LlamaIndex.

## SDK Support
Provides cross-language support including:
- Python
- Node.js
- Java
